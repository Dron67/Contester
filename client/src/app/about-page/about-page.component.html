О нас

TerCoContester - это программа для проведения турниров и индивидуального решения задач (тренировок) по спортивному программированию. TerCoContester содержит условия задач - от легких до олимпиадных - и возможность проверки решений на большинстве современных языков: C++, Object Pascal, Java и языках .NET: C#, J# и Visual Basic и др.

Авторы - Шоя Владислав Викторович и Якутов Андрей Александрович.

Большое спасибо за помощь в создании программы:
Теряеву Дмитрию Валерьевичу, Бужлакову Сергею Николаевичу и всем тем, кто ставил задачи, давал советы и тестировал.

TerCoContester написан на стеке MEAN. В TerCoContester используется сервер базы данных MongoDB.
